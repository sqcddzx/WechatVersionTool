### 说明
1. 软件会被杀毒软件误判，请关闭杀毒软件或从隔离从还原软件  
2. 首次使用前，先正常扫码确认登录一次3.9微信
3. 打开微信以及该工具  
4. 选择需要改版本的微信进程  
5. 点击修改版本，弹出修改成功即可  
6. 注意！！！ 修改版本后，第一次扫码后取消登录，再扫第二次扫码再确认登录  
（如显示 "当前客户端版本过低，请前往应用商店升级到最新版本客户端后再登录。" 直接再扫码一次登录即可）  
7. 登录完后点击修复功能
